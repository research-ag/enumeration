## Usage

### Install with mops

You need `mops` installed. In your project directory run:
```
mops add <...>
```

In the Motoko source file import the package as one of:
```
import .. "mo:..";
```

### Build & test

You need `moc` and `wasmtime` installed.
Then run:
```
git clone git@github.com:research-ag/....git
make -C test
```
